VERIFICATION

정본 확인(Verification)

정본 확인은 “진짜/가짜”만이 아닙니다. 정확한 판(Edition)과 버전(Version)을 확인해, 같은 자료를 같은 기준으로 쓰게 해줍니다.

권장 규칙: 발행일 + 버전 + 검증 코드가 모두 일치하면 정본입니다.

빠른 조회(샘플)

지금은 “샘플 화면”입니다. 실제 조회 기능(코드 입력→결과 표시)을 붙이려면 더블크로스처럼 `data`와 JS를 연결해 드릴 수 있어요.

예시 코드결과(샘플)
IPMA-PUB-DCV1-0001DOUBLE CROSS Vol.1 / First Edition / v1.0 / 정본: YES
IPMA-PUB-XXXX-0002미등록 코드(또는 오기재) / 문의 필요
